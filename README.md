# Modbus protocol implementation for master

```txt
void prerequisites(){
    download and install QtCreator; 
    // not necessarily include QSerialPort module
    if(the version u chose doesn't include QSerialPort module){
        find QT maintenance tool in the Qt folder;
        use the tool to add the QSerialPort module;
    }
    double click ModbusQT.pro to open the project;
}
```
